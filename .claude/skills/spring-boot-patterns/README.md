# spring-boot-patterns

Tham chiếu các pattern/idiom Spring Boot chuẩn của School Wallet để code mới đồng nhất: layout service, UserContext từ header gateway, gọi internal secure, orchestration giao dịch 4 pha, lock ví theo UUID, idempotency, AppException/ErrorCode, ActiveMQ event, env & cache.

Nội dung chính nằm ở [SKILL.md](SKILL.md). Gọi bằng `/spring-boot-patterns` hoặc Claude tự nạp khi viết code backend mới.
